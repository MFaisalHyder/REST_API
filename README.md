# REST_API
Spring MVC REST API with :
	
	1) MongoDB for DataBase.
	
	2) Spring MVC4 FrameWork.  
	
	3) Gradle as dependency manager.
	
	4) Spring Boot for Rapid Development.
	
	5) Angular JS for front end panel and making http calls to server.
